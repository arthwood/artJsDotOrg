ArtJs is JavaScript framework of general purpose. It allows you to select or manipulate DOM elements, perform AJAX requests, provides load of helper methods for String, Date and other native classes, etc.

For now there is no one package of ArtJs library classes. Just include files in specific order:

  • Initialize
  • utils/Log
  • events/Delegate
  • utils/MathUtils
  • utils/ObjectUtils
  • utils/ArrayUtils
  • utils/ClassUtils
  • utils/StringUtils
  • utils/DateUtils
  • utils/ElementUtils
  • events/DelegateCollection
  • events/CustomEvent
  • events/EventUtils
  • events/MouseController
  • math/Point
  • math/Rectangle
  • dom/ElementBuilder
  • dom/Selector
  • events/Clock
  • ui/DatePicker
  • ui/Flash
  • data/List
  • data/Queue
  • events/QueuedClock
  • net/Ajax
  • ui/containers/ModalBox
  • tween/Fade
  • Finalize

Some of the classes are optional (DatePicker), other are core classes on which other might depend (Delegate), but until there is no clean distinction between those two groups I recommend to include all above classes.

There are two methods that you may want to use:

ArtJs.globalize()

ArtJs.doInjection()

ArtJs.globalize() makes all the classes globally accessible in window object, as well as shorthand utility methods like: $, $$, $P, $B, $D, $DC, $E.

ArtJs.doInjection() makes an extensions on native classes like Object, Array, Date, Element etc. so that you can use:

arr.first();

otherwise you are still able to call the method using:

ArtJs.ArrayUtils.first(arr);
