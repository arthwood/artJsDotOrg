ObjectUtils

Package:

com.arthwood.utils

Description

Provides set of methods that operates on any Date instance.

Methods

  • monthDaysNum(date:Date):Number

    Returns number of days in date's month

  • firstDate(date:Date):Date

    Returns date of first day in date's month

  • firstDay(date:Date):Number

    Returns week day of first day in date's month

  • toHFM(date:Date, separator:String = ':'):String

    Returns date's string representation of hour, minute, second

  • toYMD(date:Date, separator:String = '/'):String

    Returns date's string representation of year, month, day

  • toDMY(date:Date, separator:String = '/'):String

    Returns date's string representation of day, month, year

  • toDMY(dmy:String, separator:String = '/'):Date

    Parses dmy (<day>/<month>/<year>) expression and returns Date object

  • minutesToHM(minutes:Number, separator:String = ':'):String

    Returns <hours>:<minutes> format for given amount of minutes

  • hmToMinutes(hm:String):Number

    Parses hm (<hours>:<minutes>) expression and returns Number of minutes

  • secondsToMS(seconds:Number, separator:String = ':'):String

    Returns <minutes>:<seconds> format for given amount of seconds

  • secondsToHMS(seconds:Number, separator:String = ':'):String

    Returns <hours>:<minutes>:<seconds> format for given amount of seconds

  • copy(date:Date):Date

    Returns date copy

  • getDateShifted(date:Date, days:Number):Date

    Returns date days after (before if negative) date

  • stripDayTime(date:Date):Date

    Returns date without time part
