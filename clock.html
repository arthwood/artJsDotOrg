Clock

Package:

com.arthwood.events

Description

Allows trigger events periodically

Constructor

  • Clock(interval:Number, repeat:Number)

    interval - interval between ticks in miliseconds

    repeat - number of ticks

Events

  • onChange(clock:Clock)

    Triggered on each tick

  • onFinish(clock:Clock)

    Triggered after last tick

Properties

  • interval:Number

    The same property as in constructor

  • repeat:Number

    The same property as in constructor

  • counter:Number

    Number of released ticks

Methods

  • start(now:Boolean):Void

    Starts the clock. First tick after interval or immediately if now is true

  • stop():Void

    Stops the clock and sets counter to zero

  • pause():Void

    Same as stop() but doesn't reset the counter

  • isRunning():Void

    Returns true if clock is running; false otherwise
