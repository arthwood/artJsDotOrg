Clock

Package:

com.arthwood.events

Description

Allows trigger events periodically

Constructor

  • Clock(interval:Number, repeat:Number)

    interval - interval between ticks in miliseconds

    repeat - number of ticks

    Example:

    var clock = new Clock(1000, 5);

Events

  • onChange(clock:Clock)

    Triggered on each tick

    Example:

    var clock = new Clock(1000, 5);

    clock.onChange.add(new Delegate(this, this.onClockChange);

    function onClockChange(clock) {
      alert('clock change!');
    }

  • onFinish(clock:Clock)

    Triggered after last tick

    Example:

    var clock = new Clock(1000, 5);

    clock.onComplete.add(new Delegate(this, this.onClockComplete);

    function onClockComplete(clock) {
      alert('clock complete!');
    }

Properties

  • interval:Number

    The same property as in constructor

  • repeat:Number

    The same property as in constructor

  • counter:Number

    Number of released ticks

Methods

  • start(now:Boolean):Void

    Starts the clock. First tick after interval or immediately if now is true

    Example:

    var clock = new Clock(1000, 5);

    clock.start();

  • stop():Void

    Stops the clock and sets counter to zero

    Example:

    var clock = new Clock(1000, 5);

    clock.start();
    // ... and some time later
    clock.counter; // 3
    clock.stop();
    clock.counter; // 0

  • pause():Void

    Same as stop() but doesn't reset the counter

    Example:

    var clock = new Clock(1000, 5);

    clock.start(true);
    // ... and some time later
    clock.counter; // 3
    clock.pause();
    clock.counter; // 3

  • resume():Void

    Resumes ticking after it has been paused

    Example:

    var clock = new Clock(1000, 5);

    clock.start();
    // ... and some time later
    clock.counter; // 3
    clock.pause();
    // ... and some time later
    clock.resume();
    clock.counter; // 3

  • isRunning():Void

    Returns true if clock is running; false otherwise

    Example:

    var clock = new Clock(1000, 5);

    clock.isRunning(); // false
    clock.start(true);
    clock.isRunning(); // true
    clock.stop();
    clock.isRunning(); // false
