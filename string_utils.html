StringUtils

Package:

com.arthwood.utils

Description

Provides set of methods that operates on any String instance.

Methods

  • first(string:String):String

    Returns first letter of a string

  • last(string:String):String

    Returns last letter of a string

  • stripSpaces(string:String):String

    Removes all spaces from string

  • stripNewLines(string:String):String

    Removes all new line characters from string

  • stripTabs(string:String):String

    Removes all tabs from string

  • blank(string:String):Boolean

    Returns true if string is null or empty; false otherwise

  • empty(string:String):Boolean

    Returns true if string is empty; false otherwise

  • nullifyEmpty(string:String):Object

    Returns null if string is empty; string otherwise

  • toS(string:String):String

    Returns empty string if string is null; string otherwise

  • replace(string:String, pattern:String, substitute:String):String

    Replaces all occurences of pattern in string with substitute

  • countPattern(string:String, pattern:String):Number

    Returns number of pattern occurences in string

  • align(string:String, length:Number, char:String, left:Boolean):String

    Extends length of the string to length value by filling the gap with char characters on the side determined by left argument

  • getMultiPattern(pattern:String, n:Number):String

    Returns string containing of pattern concatenated n times

  • formatPrice(value:Number):String

    Returns price formatted string

  • truncate(string:String, n:Number, end:String = '...'):String

    Truncates string to length n and appends end

  • singularOrPlural(string:String, n:Number):String

    Returns plural version of string if n is other than 1

  • capitalize(string:String):String

    Upcase each word in string

  • trim(string:String):String

    Remove all the whitespaces from the begining and the end of the string

  • sub(string:String, start:Number, end:Number):String

    Periodic version of String.substring()

  • toJson(string:String):Object

    Converts string JSON format to Object
