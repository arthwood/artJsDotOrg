List

Package:

com.arthwood.data

Description

List data model implementation

Constructor

  • List(array:Array = [])

    array - collection data

    Example:

    var list = new List([1, 2, 3, 4, 5]);

Events

  • onChange(list:List)

    Triggered every time the collection changes

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.onChange.add(new ArtJs.Delegate(this, this.onChange));
    function onChange(list) { alert('list has changed!'); }

Properties

  • allowDuplicates:Boolean = true

    Flag that allows having duplicates in collection

    If this flag is set to true, adding existing item won't do anything.

  • loop:Boolean = false

    Flag that allows periodically return the item, even if index is out of collection bounds

    If this flag is set to true and you try to fetch an item by index out of collection bounds, you will receive item as if you were fetching it from infinite collection consisting of periodically repeated items of initial collection.

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.loop = true;

    list.getItemAt(-1); //5
    list.getItemAt(6); //2

Methods

  • addItem(item:Object, noEvent:Boolean):Number

    Adds item to the end of collection and returns new length.
    If noEvent is set to true, this action wont dispatch the change event.

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.addItem(6); //6
    list.toString() // 1, 2, 3, 4, 5, 6

  • addItemAt(item:Object, idx:Number, noEvent:Boolean):Void

    Adds item to collection at specific position and returns new length

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.addItemAt(6, 2); //6
    list.toString() // 1, 2, 6, 3, 4, 5

  • removeItem(item:Object, onlyFirst:Boolean, noEvent:Boolean):Void

    Removes item from collection

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.removeItem(2); //4
    list.toString() // 1, 3, 4, 5

  • removeItemAt(idx:Number, noEvent:Boolean):Void

    Removes item at specific position from collection

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.removeItemAt(2); //4
    list.toString() // 1, 2, 4, 5

  • removeAll(noEvent:Boolean):Void

    Removes all items from collection

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.removeAll();
    list.toString() //

  • getItemAt(idx:Number):Object

    Retrieves item at specific position

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.getItemAt(3); // 4

  • getItemIndex(item:Object):Number

    Retrieves item's index

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.getItemIndex(3); // 2

  • moveItem(fromIndex:Number, toIndex:Number):Void

    Moves item from, to specific position

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.moveItem(3, 0);
    list.toString(); // 4, 1, 2, 3, 5;

  • getLength():Number

    Returns length of collection

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.getLength(); // 5

  • getItems():Array

    Returns copy of collection

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.getItems().toString(); // 1, 2, 3, 4, 5

  • setItems(items:Array):Void

    Set the collection

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.toString(); // 1, 2, 3, 4, 5
    list.setItems([4, 5, 6]);
    list.toString(); // 4, 5, 6

  • hasItem(item:Object):Boolean

    Returns true if item is in collection; false otherwise

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.hasItem(4); // true
    list.hasItem(7); // false

  • setPointerAtItem(item:Object):Void

    Set the pointer at position of item

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.setPointerAtItem(2);
    list.getPointer(); // 1

  • reset():Void

    Sets pointer to zero

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.setPointer(2);
    list.getPointer(); // 2
    list.reset();
    list.getPointer(); // 0

  • getPointer():Number

    Returns pointer

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.getPointer(); // 0
    list.increase();
    list.getPointer(); // 1

  • setPointer(i:Number):Void

    Sets pointer

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.setPointer(2);
    list.getPointer(); // 2

  • decrease():Void

    Decreases pointer

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.setPointer(2);
    list.decrease();
    list.getPointer(); // 1

  • increase():Void

    Increases pointer

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.setPointer(2);
    list.increase();
    list.getPointer(); // 3

  • getCurrent():Object

    Returns current item

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.setPointer(2);
    list.getCurrent(); // 3

  • getPrevious():Object

    Returns item before current

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.setPointer(2);
    list.getPrevious(); // 2

  • getNext():Object

    Returns item after current

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.setPointer(2);
    list.getNext(); // 4

  • getFirst():Object

    Returns first item of collection

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.getFirst(); // 1

  • getLast():Object

    Returns last item of collection

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.getLast(); // 5

  • empty():Boolean

    Returns true if collection is empty; false otherwise

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.empty(); // false
    list.removeAll();
    list.empty(); // true

  • isLast():Boolean

    Returns true if pointer is set to last item in collection; false otherwise

    Example:

    var list = new List([1, 2, 3, 4, 5]);

    list.setPointer(4);
    list.isLast(); // true
