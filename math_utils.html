MathUtils

Package:

com.arthwood.utils

Description

Provides set of math functions.

Methods

  • sgn(x:Number):Number

    Signum function of x

    Example:

    MathUtils.sgn(-2); // -1
    MathUtils.sgn(0); // 0
    MathUtils.sgn(15); // 1

  • limit(x:Number, a:Number, b:Number):Number

    Returns x value limited to [a, b] range

    Example:

    MathUtils.limit(6, 2, 13); // 6
    MathUtils.limit(6, 8, 13); // 8
    MathUtils.limit(6, -2, 3); // 3

  • periodicLimit(x:Number, a:Number, b:Number):Number

    Returns x value periodically limited to [a, b] range

    Example:

    MathUtils.periodicLimit(6, 2, 13); // 6
    MathUtils.periodicLimit(6, 8, 13); // 11
    MathUtils.periodicLimit(6, -2, 3); // 1

  • stairs(x:Number, a:Number, b:Number):Number

    "stairs" function of x

    Example:

    MathUtils.stairs(6, 2, 4); // 2
    MathUtils.stairs(6, 1, 2); // 5
    MathUtils.stairs(6, -1, 1); // 3
