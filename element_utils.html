ElementUtils

Package:

com.arthwood.utils

Description

Provides set of methods that operates on any Element instance.

Methods

  • show(element:Element):Void

    Shows the element

  • hide(element:Element):Void

    Hides the element

  • toggle(element:Element):Void

    Shows the element if it's hidden; hides otherwise

  • setVisible(element:Element, visible:Boolean):Void

    Shows the element if visible argument is set to true; hides otherwise

  • isHidden(element:Element):Boolean

    Returns true if the element is hidden; true otherwise

  • setAlpha(element:Element, value:Number):Void

    Sets element's transparency; value 0 means transparent, 1 means opaque

  • getSize(element:Element):Point

    Returns size of the element as a com.arthwood.math.Point object which x property is width and y property is height

  • elements(element:Element):Array

    Returns array of immediate descendants (Elements) of element

  • isElement(element:Element):Boolean

    Returns true if element is Element; false otherwise

  • remove(element:Element):Node

    Removes the element

  • parent(element:Element):Element

    Returns element's parent element

  • firstElement(element:Element):Element

    Returns element's first child (Element)

  • prev(element:Element):Node

    Returns element's previous sibling (Element)

  • next(element:Element):Node

    Returns element's next sibling (Element)

  • clone(element:Element, deep:Boolean):Node

    Alias for Node.cloneNode(deep)

  • putAtBottom(element:Element, ref:Node):Node

    Inserts clone of the element as the last child of ref node

  • putAtTop(element:Element, ref:Node):Node

    Inserts clone of the element as the first child of ref node

  • putAfter(element:Element, ref:Node):Node

    Inserts clone of the element as next sibling of ref node

  • putBefore(element:Element, ref:Node):Node

    Inserts clone of the element as previous sibling of ref node

  • replace(element:Element, ref:Node, clone:Boolean):Node

    Replaces ref with an element (or its clone if clone is set to true)

  • center(element:Element):Void

    Centers element on the screen

  • centerH(element:Element):Void

    Centers element horizontally on the screen

  • centerV(element:Element):Void

    Centers element vertically on the screen

  • getCenteredPosition(element:Element):Point

    Returns position of element as if it was centered on the screen

  • getDocumentSize():Point

    Returns document size

  • getWindowSize():Point

    Returns window size

  • setPosition(element:Element, position:Point):Void

    Sets the position of element on the screen

  • getPosition(element:Element, withoutScroll:Boolean):Point

    Returns position of the element on the screen

  • getScrollPosition():Point

    Returns scroll (x, y) values

  • setX(element:Element, value:Number):Void

    Sets x position of element on the screen

  • setY(element:Element, value:Number):Void

    Sets y position of element on the screen

  • getLayout(element:Element, withoutScroll:Boolean):Rectangle

    Returns position and size of element as a (x, y, width, height) Rectangle

  • enable(element:Element):Void

    Enables the element

  • disable(element:Element):Void

    Disables the element

  • setEnabled(element:Element, enabled:Boolean):Void

    Enables element if enabled is true; disables otherwise
