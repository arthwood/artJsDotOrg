ObjectUtils

Package:

com.arthwood.utils

Description

Provides set of methods that operates on any Object instance.

Methods

  • copy(object:Object):Object

    Creates a copy of an object.

  • copyProps(source:Object, target:Object):Void

    Copies properties from source object to taraget object.

  • extend(target:Object, base:Object):Void

    Copies properties from base object to target object.

  • merge(target:Object, base:Object):Object

    Works the same as extend() but returns modified target object.

  • removeValue(object:Object, value:Object):Void

    Removes all object's properties with value equal to value param

  • removeValues(object:Object, values:Array):Void

    Works the same as removeValue but you can remove many values by passing them in array

  • map(object:Object, callback:Function):Array

    Maps key and value to Array item

  • mapValue(object:Object, callback:Function):Array

    Maps value to Array item

  • mapKey(object:Object, callback:Function):Array

    Maps key to Array item

  • each(object:Object, callback:Function):Void

    Iterates over properties and passes value to callback

  • eachKey(object:Object, callback:Function):Void

    Iterates over properties and passes key to callback

  • eachPair(object:Object, callback:Function):Void

    Iterates over properties and passes key and value to callback

  • select(object:Object, callback:Function):Object

    Returns copy of object containing only properties for which callback returns true

  • selectWithKey(object:Object, callback:Function):Object

    Same as select, but callback receives both key and value

  • reject(object:Object, callback:Function):Object

    Returns copy of object without properties for which callback returns true

  • empty(object:Object):Boolean

    Returns true if object has no properties; false otherwise

  • fromArray(array:Array):Object

    Transforms array into object

  • toArray(object:Object):Array

    Transforms object into array

  • include(object:Object, item:Object):Boolean

    Returns true if any object value is equal to item; false otherwise

  • includeAll(object:Object, subset:Object):Boolean

    Returns true if all subset values can be found among object's values; false otherwise

  • toQueryString(object:Object):String

    Returns query string representation of object
