ArtJs

Description

This is top level object that stores information about framework and few methods that allows you to select the framework mode

Constants

  • VERSION:String

    Current version of framework

  • CLIENT:String

    Dedicated browser

Methods

  • globalize():Void

    Makes all classes accessible globally in window object

    Normally after framework is loaded you have two ways of accessing its classes:

    - using full qualified path (com.arthwood.utils.ArrayUtils)
    - by looking into top ArtJs namespace (ArtJs.ArrayUtils)

    For now there is no classes with the same name in whole framework so accessing it via ArtJs is fine.
    However, if at some point it happens, full qualified name is necessary.
    When all framework classes are loaded and accessible (e.g. in js file following the framework file) simply use:

    ArtJs.globalize();

  • doInjection():Void

    After calling this method you are able to call Utils methods on native objects

    Some of the class methods from framework can get injected into native classes.
    For example most of com.arthwood.utils.ArrayUtils would then affect native Array object.
    To achieve this, when all framework classes are accessible use:

    ArtJs.doInjection();

    After that action you will be able to write:

    myArray.last(); // 3

    instead of:

    ArtJs.ArrayUtils.last(myArray); // 3

    To see what framework functionality is affecting native classes see other parts of documentation.

  • $():Element

    Alias for ElementUtils.getElementById()

    Example:

    var nav = ArtJs.$('navigation');

  • $$():Array

    Alias for ElementUtils.getElements()

    Example:

    var nav = ArtJs.$$('.main p.item span');

  • $down():Array

    Alias for ElementUtils.down()

    Example:

    var nav = ArtJs.$('navigation');
    var items = ArtJs.$down(nav, '.item');

  • $up():Element

    Alias for ElementUtils.up()

    Example:

    var nav = ArtJs.$('navigation');
    var parent = ArtJs.$up(nav);

  • $P():Element

    Alias for ElementBuilder.parse()

    Example:

    var element = ArtJs.$P('Blue t-shirt');

  • $get():Ajax

    Alias for Ajax.get()

    Example:

    var ajax = ArtJs.$get('http://mydomain.com', null, new Delegate(this, this.onAjaxSuccess));

  • $post():Ajax

    Alias for Ajax.post()

    Example:

    var ajax = ArtJs.$post('http://mydomain.com', {id: 4}, new Delegate(this, this.onAjaxSuccess));

  • $put():Ajax

    Alias for Ajax.$put()

    Example:

    var ajax = ArtJs.$put('http://mydomain.com', {id: 4}, new Delegate(this, this.onAjaxSuccess));

  • $del():Ajax

    Alias for Ajax.del()

    Example:

    var ajax = ArtJs.$del('http://mydomain.com', {id: 4}, new Delegate(this, this.onAjaxSuccess));

  • p():Void

    Debug info. If debug console is accessible output to the console; uses alert() otherwise.

    Example:

    p('myVariable: ' + myVariable);

  • $D():Delegate

    Alias for Delegate.create()

    Example:

    var delegate = ArtJs.$D(this, this.onClick);

  • $DC():Function

    Alias for Delegate.callback()

    Example:

    var callback = ArtJs.$DC(this, this.onClick, true);
