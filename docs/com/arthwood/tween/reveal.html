Reveal

Package:

com.arthwood.tween

Description

Implements "reveal" effect

Constructor

  • Reveal(element:Element, delta:Number = 0.05, interval:Number = 20)

    element - element on which effect is applied

    delta - increment (0 < delta < 1)

    interval - animation timeframe in miliseconds

    Example:

    var reveal = new Reveal($('nav'), 0.2, 30);

Events

  • onComplete(reveal:Reveal)

    Dispatched after effect finish

    Example:

    var reveal = new Reveal($('nav'), 0.2, 30);

    reveal.onComplete.add(new Delegate(null, onComplete));

    reveal.start();

    function onComplete(reveal) {
      alert('animation completed!');
    }

Properties

  • delta:Number

    The same property as in constructor

  • element:Element

    The same property as in constructor

  • interval:Number

    The same property as in constructor

Methods

  • isRunning():Boolean

    Indicates if animation is in progress

    Example:

    var reveal = new Reveal($('nav'), 0.2, 30);

    reveal.start();

    reveal.isRunning(); // true

  • start():Void

    Starts the animation

    Example:

    var reveal = new Reveal($('nav'), 0.2, 30);

    reveal.start();

  • stop():Void

    Stops the animation

    Example:

    var reveal = new Reveal($('nav'), 0.2, 30);

    reveal.start();

    // ... some time later
    reveal.stop();
