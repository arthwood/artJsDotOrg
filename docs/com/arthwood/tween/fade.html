Fade

Package:

com.arthwood.tween

Description

Implements "fade" effect

Constructor

  • Fade(element:Element, delta:Number = 0.05, interval:Number = 20)

    element - element on which effect is applied

    delta - increment (0 < delta < 1)

    interval - animation timeframe in miliseconds

    Example:

    var fade = new Fade($('nav'), 0.2, 30);

Events

  • onComplete(fade:Fade)

    Dispatched after effect finish

    Example:

    var fade = new Fade($('nav'), 0.2, 30);

    fade.onComplete.add(new Delegate(null, onComplete));

    fade.start();

    function onComplete(fade) {
      alert('animation completed!');
    }

Properties

  • delta:Number

    The same property as in constructor

  • element:Element

    The same property as in constructor

  • interval:Number

    The same property as in constructor

Methods

  • isRunning():Boolean

    Indicates if animation is in progress

    Example:

    var fade = new Fade($('nav'), 0.2, 30);

    fade.start();

    fade.isRunning(); // true

  • start():Void

    Starts the animation

    Example:

    var fade = new Fade($('nav'), 0.2, 30);

    fade.start();

  • stop():Void

    Stops the animation

    Example:

    var fade = new Fade($('nav'), 0.2, 30);

    fade.start();

    // ... some time later
    fade.stop();
