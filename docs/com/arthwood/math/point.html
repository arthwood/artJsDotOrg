Point

Package:

com.arthwood.math

Description

Math vector implementation

Constructor

  • Point(x:Number, y:Number)

    x - x coordinate

    y - y coordinate

    Example:

    var point = new Point(20, 40);

Properties

  • x:Number

    The same property as in constructor

  • y:Number

    The same property as in constructor

Methods

  • add():Point

    Adds other vector

    Example:

    var p1 = new Point(2, 3);
    var p2 = new Point(4, -1);

    p1.add(p2); // (6, 2)

  • dot():Number

    Dot product of this vector and other vector

    Example:

    var p1 = new Point(2, 3);
    var p2 = new Point(4, -1);

    p1.dot(p2); // 5

  • getLength():Number

    Length of a vector

    Example:

    var point = new Point(3, 4);

    point.getLength(); // 5

  • getReversed():Point

    Returns reversed vector

    Example:

    var p = new Point(2, 3);

    p.getReversed(); // (-2, -3)
    p; // (2, 3)

  • getTransposed():Point

    Returns new transposed vector

    Example:

    var p = new Point(2, 3);

    p.getTransposed(); // (3, 2)
    p; // (2, 3)

  • reverse():Point

    Reverses vector and returns this vector

    Example:

    var p = new Point(2, 3);

    p.reverse(); // (-2, -3)
    p; // (-2, -3)

  • reverseX():Point

    Reverses x coordinate and returns this vector

    Example:

    var p = new Point(2, 3);

    p.reverse(); // (-2, 3)
    p; // (-2, 3)

  • reverseY():Point

    Reverses y coordinate and returns this vector

    Example:

    var p = new Point(2, 3);

    p.reverse(); // (2, -3)
    p; // (2, -3)

  • sub():Void

    Subtracts other vector

    Example:

    var p1 = new Point(2, 3);
    var p2 = new Point(4, -1);

    p1.sub(p2); // (-2, 4)

  • times(k:Number):Point

    Returns new vector multiplied by k

    Example:

    var p = new Point(2, 3);

    p.times(-3); // (-9, -6)
    p; // (2, 3)

  • toString():String

    Returns string representation od vector

    Example:

    var p = new Point(2, 3);

    p.toString(); // [2, 3]

  • transpose():Point

    Transposes coordinates and returns this vector

    Example:

    var p = new Point(2, 3);

    p.transpose(); // (3, 2)
    p; // (3, 2)
