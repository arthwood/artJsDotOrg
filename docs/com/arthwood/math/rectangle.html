Rectangle

Package:

com.arthwood.math

Description

Math rectangle implementation

Constructor

  • Rectangle(left:Number, top:Number, right:Number, bottom:Number)

    left - left edge

    top - top edge

    right - right edge

    bottom - bottom edge

    Example:

    var rectangle = new Rectangle(10, 10, 30, 20);

Properties

  • bottom:Number

    The same property as in constructor

  • left:Number

    The same property as in constructor

  • right:Number

    The same property as in constructor

  • top:Number

    The same property as in constructor

Methods

  • getArea():Number

    area of rectangle

    Example:

    var rectangle = new Rectangle(10, 10, 30, 20);

    rectangle.getArea(); // 200

  • getHeight():Number

    height of rectangle

    Example:

    var rectangle = new Rectangle(10, 10, 30, 20);

    rectangle.getHeight(); // 10

  • getLeftBottom():Point

    left-bottom corner vector

    Example:

    var rectangle = new Rectangle(10, 10, 30, 20);

    rectangle.getRightBottom(); // [10, 20]

  • getLeftTop():Point

    left-top corner vector

    Example:

    var rectangle = new Rectangle(10, 10, 30, 20);

    rectangle.getLeftTop(); // [10, 10]

  • getRightBottom():Point

    right-bottom corner vector

    Example:

    var rectangle = new Rectangle(10, 10, 30, 20);

    rectangle.getRightBottom(); // [30, 20]

  • getRightTop():Point

    right-top corner vector

    Example:

    var rectangle = new Rectangle(10, 10, 30, 20);

    rectangle.getRightTop(); // [30, 10]

  • getSize():Point

    width and height as a vector

    Example:

    var rectangle = new Rectangle(10, 10, 30, 20);

    rectangle.getSize(); // [20, 10]

  • getWidth():Number

    width of rectangle

    Example:

    var rectangle = new Rectangle(10, 10, 30, 20);

    rectangle.getWidth(); // 20

  • moveBy(point:Point):Void

    Moves rectangle by a vector

    Example:

    var rectangle = new Rectangle(10, 10, 30, 20);

    rectangle.moveBy(new Point(-10, -10);
    rectangle.toString(); // [0, 0, 20, 10]
