Selector

Package:

com.arthwood.dom

Description

Allows to easily traverse DOM

Methods

  • descendantOf(element:Element, root:Element):Boolean

    Returns true if element is descendant of root

    Example:

    var element = $('menu');
    Selector.descendantOf(element, $('container'));
    // or
    // element.descendantOf($('container'));

  • down(element:Element, path:String):Array

    Returns array of elements that meets the path requirements

    Example:

    var element = $('menu');
    var nav = Selector.down(element, '.nav');
    // or
    // var nav = element.down('.nav');

  • getFamily(element:Element, root:Element = body):Array

    If element is descendant of root it returns array consisting of element and it's ascendants; otherwise returns empty array.

    Example:

    var element = $('menu');
    var family = Selector.getFamily(element, $('container'));
    // or
    // var family = element.getFamily($('container'));

  • up(element:Element, path:String):Element

    Returns first ascendant of element that meets the path requirements

    Example:

    var element = $('nav');
    var menu = Selector.up(element, '.menu');
    // or
    // var menu = element.up('.menu');
