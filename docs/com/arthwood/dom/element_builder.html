ElementBuilder

Package:

com.arthwood.dom

Description

Allows to easily create DOM elements

Constructor

  • ElementBuilder(name:String, attributes:Object, value:String, empty:Boolean)

    name - element name

    attributes - element attributes

    value - element inner text

    empty - if true, empty element will be created

    Example:

    var builder = new ElementBuilder('span', {class: 'price'}, '$199');

Properties

  • attributes:Object

    The same property as in constructor

  • empty:Boolean

    The same property as in constructor

  • name:String

    The same property as in constructor

  • value:Object

    The same property as in constructor

Methods

  • getElement():Node

    Builds and returns new Element

    Example:

    var builder = new ElementBuilder('span', {class: 'price'}, '$199');
    var element = builder.getElement();

    element.toString(); // [object HTMLSpanElement]

  • toString():String

    String representation of element

    Example:

    var builder = new ElementBuilder('span', {class: 'price'}, '$199');

    builder.toString(); // <span class="price">$199</span>

Static methods

  • parse(string:String):Node

    Parses string and returns Node

    Example:

    var element = ElementBuilder.parse('<span class="price">$199</span>');

    element.toString(); // [object HTMLSpanElement]
