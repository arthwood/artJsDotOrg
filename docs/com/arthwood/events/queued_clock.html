QueuedClock

Package:

com.arthwood.events

Description

Allows trigger events periodically

Constructor

  • QueuedClock(interval:Number)

    interval - interval between clock ticks in miliseconds

    The class works like queue. You add Delegate items and then clock periodically removes current item and calls invoke() on it.

    Example:

    var qc = new QueuedClock(1000);

    // add one delegate item
    qc.addItem(new Delegate(this, this.onTick)) // 1;
    qc.getLength(); // 1
    qc.start();
    // after first tick the collection is empty
    qc.getLength(); // 0

Properties

  • interval:Number

    The same property as in constructor

  • busy:Boolean

    Indicates if clock is running

Methods

  • addItem(item:Delegate):Number

    Adds delegate to collection and returns its new length

    Example:

    var qc = new QueuedClock(1000);

    qc.addItem(new Delegate(this, this.onTick)) // 1;

  • start():Void

    Starts execution

    Example:

    var qc = new QueuedClock(1000);

    qc.addItem(new Delegate(this, this.onTick)); qc.start();

  • getLength():Number

    Returns current length of collection

    Example:

    var qc = new QueuedClock(1000);

    qc.getLength(); // 0
    qc.addItem(new Delegate(this, this.onTick));
    qc.getLength(); // 1
