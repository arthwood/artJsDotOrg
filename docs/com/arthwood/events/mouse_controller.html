MouseController

Package:

com.arthwood.events

Description

Allows do detect mouse events on edge of an element

Constructor

  • MouseController(element:Element)

    element - controlled element

    Example:

    var mc = new MouseController(element);

    mc.onOver.add(new Delegate(this, onOver));
    mc.onOut.add(new Delegate(this, onOut));

Events

  • onOut(mc:MouseController, e:Event)

    Dispatched after mouse leaves the element area

    Example:

    var mc = new MouseController(element);

    mc.onOut.add(new Delegate(this, onOut));

  • onOver(mc:MouseController, e:Event)

    Dispatched after mouse enters the element area

    Example:

    var mc = new MouseController(element);

    mc.onOver.add(new Delegate(this, onOver));

Properties

  • element:Element

    The same property as in constructor

  • over:Boolean

    Indicates if mouse is currently over the element

Methods

  • getTargets(event:Event, over:Boolean):Object

    event - event object

    over - pass true if you call this action in mouse over event handler; false otherwise

    Returns object containing information about the event.
    This object contains following properties:
    - origin
    - current
    - related

    Example:

    var mc = new MouseController(element);

    mc.onOver.add(new Delegate(this, onOver));

    function onOver(e, mc) {
      var targets = mc.getTargets(e, true);

      if (targets.related == someElement) {
        // do something
      }
    }
