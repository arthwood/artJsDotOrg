CustomEvent

Package:

com.arthwood.events

Description

Defines object capable of dispatching events.

Constructor

  • CustomEvent(name:String = null)

    name - name of the event

    Example:

    var myEvent = new CustomEvent('MyClass::myEvent');

Properties

  • name:Name

    The same property as in constructor

  • collection:DelegateCollection

    Collection of registered listeners

    Example:

    var myEvent = new CustomEvent('MyClass::myEvent');

    myEvent.collection.length // 0
    myEvent.add(new Delegate(null, function() {
      alert('event!')
    }));
    myEvent.collection.length // 1

Methods

  • add(delegate:Delegate):Void

    Registers listener

    Example:

    var myEvent = new CustomEvent('MyClass::myEvent');

    myEvent.add(new Delegate(null, function() {
      alert('event!')
    }));

  • fire():Array

    Triggers the event and returns responses from handlers as an array

    Example:

    var myEvent = new CustomEvent('MyClass::myEvent');

    myEvent.add(new Delegate(null, function() {
      alert('event!');
      return 2;
    }));

    // ...then at some point you dispatch the event
    myEvent.fire(this, 'hello'); // [2]

  • getLength():Number

    Amount of listeners currently registered

    Example:

    var myEvent = new CustomEvent('MyClass::myEvent');
    var delegateOne = new Delegate(null, function(msg) {
      alert('event says to delegate 1: ' + msg);
    });
    var delegateTwo = new Delegate(null, function(msg) {
      alert('event says to delegate 2: ' + msg);
    });

    myEvent.getLength(); // 0
    myEvent.add(delegateOne);
    myEvent.getLength(); // 1
    myEvent.add(delegateTwo);
    myEvent.getLength(); // 2
    myEvent.removeAll();
    myEvent.getLength(); // 0

  • remove(delegate:Delegate):Void

    Removes listener

    Example:

    var myEvent = new CustomEvent('MyClass::myEvent');
    var delegate = new Delegate(null, function(msg) {
      alert('event says: ' + msg)
    });

    myEvent.add(delegate);
    myEvent.fire('hello!'); // delegate handles event
    myEvent.remove(delegate);
    myEvent.fire('hello!'); // nothing happens

  • removeAll():Void

    Removes all listeners

    Example:

    var myEvent = new CustomEvent('MyClass::myEvent');
    var delegateOne = new Delegate(null, function(msg) {
      alert('event says to delegate 1: ' + msg);
    });
    var delegateTwo = new Delegate(null, function(msg) {
      alert('event says to delegate 2: ' + msg);
    });

    myEvent.add(delegateOne);
    myEvent.add(delegateTwo);
    myEvent.fire('hello!'); // delegates handle event
    myEvent.removeAll();
    myEvent.fire('hello!');// nothing happens
