DatePicker

Package:

com.arthwood.ui

Description

Calendar date picker

When instantiating DatePicker class it automatically searches for elements with '.datepicker' class and make them datepicker elements. However when the field is loaded on runtime you need to activate it manually using initField(input) method.

Constructor

  • DatePicker(firstDay:Number)

    firstDay - first day of the week displayed in calendar; 0 is for Sunday

    Example:

    var datePicker = new DatePicker(1);

Methods

  • initField(input:HTMLInputElement):Void

    Initializes input element so it becomes datepicker input

    Example:

    datePicker.initField(input);
