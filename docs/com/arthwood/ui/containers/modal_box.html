ModalBox

Package:

com.arthwood.ui.containers

Description

Display window with loadable content

Constructor

  • ModalBox(element:Element, size:Point)

    element - modalbox element

    size - size of the modal box

    Example:

    var modalBox = ModalBox($('nav'), new Point(300, 400));

Methods

  • setTitle(title:String):Void

    Sets header title

    Example:

    var modalBox = ModalBox($('nav'), new Point(300, 400));

    modalBox.setTitle('Hotel details');

  • show(url:String):Void

    Shows modal box and loads content

    Example:

    var modalBox = ModalBox($('nav'), new Point(300, 400));

    modalBox.show('/hotels/3');
