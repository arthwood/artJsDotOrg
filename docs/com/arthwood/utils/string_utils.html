StringUtils

Package:

com.arthwood.utils

Description

Provides set of methods that operates on any String instance.

Methods

  • align(string:String, length:Number, char:String, left:Boolean):String

    Extends length of the string to length value by filling the gap with char characters on the side determined by left argument

    Example:

    var string = 'color';

    StringUtils.align(string, 10, '-', true); // color-----
    // or
    // string.align(10, '-', true);

  • blank(string:String):Boolean

    Returns true if string is null or empty; false otherwise

    Example:

    var string = 'quantum mechanics';
    var emptyString = ' ';
    var nullValue = null;

    StringUtils.blank(string); // false
    // or
    // string.blank();

    StringUtils.blank(emptyString); // true
    // or
    // emptyString.blank();

    StringUtils.blank(nullValue); // true

  • capitalize(string:String):String

    Upcase each word in string

    Example:

    var string = 'advanced quantum mechanics';

    StringUtils.capitalize(string); // Advanced Quantum Mechanics
    // or
    // string.capitalize();

  • countPattern(string:String, pattern:String):Number

    Returns number of pattern occurences in string

    Example:

    var string = 'advanced quantum mechanics';

    StringUtils.countPattern(string, 'an'); // 3
    // or
    // string.countPattern('an');

  • empty(string:String):Boolean

    Returns true if string is empty; false otherwise

    Example:

    var string = 'quantum mechanics';
    var emptyString = ' ';

    StringUtils.empty(string); // false
    // or
    // string.empty();

    StringUtils.empty(emptyString); // true
    // or
    // emptyString.empty();

  • first(string:String):String

    Returns first letter of a string

    Example:

    var string = 'quantum mechanics';

    StringUtils.first(string); // q
    // or
    // string.first();

  • formatPrice(value:Number):String

    Returns price formatted string

    Example:

    var price = 3.7;

    StringUtils.formatPrice(price); // 3.70

  • getMultiPattern(pattern:String, n:Number):String

    Returns string containing of pattern concatenated n times

    Example:

    var string = 'yes, ';

    StringUtils.getMultiPattern(string, 3); // yes, yes, yes,
    // or
    // string.getMultiPattern(3);

  • last(string:String):String

    Returns last letter of a string

    Example:

    var string = 'quantum mechanics';

    StringUtils.last(string); // s
    // or
    // string.last();

  • nullifyEmpty(string:String):Object

    Returns null if string is empty; string otherwise

    Example:

    var string = 'quantum mechanics';
    var emptyString = ' ';

    StringUtils.nullifyEmpty(string); // quantum mechanics
    // or
    // string.nullifyEmpty();

    StringUtils.nullifyEmpty(emptyString); // null
    // or
    // emptyString.nullifyEmpty();

  • singularOrPlural(string:String, n:Number):String

    Returns plural version of string if n is other than 1

    Example:

    var string = 'chicken';

    StringUtils.singularOrPlural(string, 1); // chicken
    // or
    // string.singularOrPlural(1);

    StringUtils.singularOrPlural(string, 5); // chickens
    // or
    // string.singularOrPlural(5);

  • strip(string:String):String

    Removes whitespaces from string and returns the result

    Example:

    var string = 'quantum mechanics';

    StringUtils.strip(string); // quantummechanics
    // or
    // string.strip();

  • sub(string:String, start:Number, end:Number):String

    Periodic version of String.substring()

    Example:

    var string = 'quantum mechanics';

    StringUtils.sub(string, -2, 4); // csquan
    // or
    // string.sub(-2, 4);

  • toJson(string:String):Object

    Converts string JSON format to Object

    Example:

    var string = "{id: 5, name: 'Mike'}";

    StringUtils.toJson(string); // {id: 5, name: 'Mike'}
    // or
    // string.toJson();

  • toS(string:String):String

    Returns empty string if string is null; string otherwise

    Example:

    var string = 'quantum mechanics';
    var nullValue = null;

    StringUtils.toS(string); // quantum mechanics
    StringUtils.toS(nullValue); // (empty string)

  • trim(string:String):String

    Remove all the whitespaces from the begining and the end of the string

    Example:

    var string = ' quantum mechanics ';

    StringUtils.trim(string); // quantum mechanics
    // or
    // string.trim();

  • truncate(string:String, n:Number, end:String = '...'):String

    Truncates string to length n and appends end

    Example:

    var string = 'advanced quantum mechanics';

    StringUtils.truncate(string, 8); // advanced...
    // or
    // string.truncate(8);
