EventUtils

Package:

com.arthwood.utils

Description

Provides set of methods related to event handling.

Methods

  • edge(targets:Object):Boolean

    Returns true if mouse event occured on the edge on an Element; false otherwise

    Example:

    var mc = new MouseController(element);

    // in mouseover event handler
    EventUtils.edge(element, mc.getTargets(event, true));

    // or
    // element.edge(mc.getTargets(event, true));
