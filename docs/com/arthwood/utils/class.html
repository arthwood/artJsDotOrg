Class

Package:

com.arthwood.utils

Description

Facilitates class inheritance.

Methods

  • extend(base:Function, constr:Function, instanceMethods:Object, classMethods:Object):Function

    Creates a subclass ob base class

    Example:

    Animal = ArtJs.Class( function(name) { this.name = name; }, { getSound: function() { return '...'; } }, getName: function() { return 'animal'; } ); Dog = ArtJs.Class( function(name, options) { this.super(arguments, name); this.options = options; }, { getSound: function() { var sound = this.super(arguments); return sound + 'meoww'; } }, { getName: function() { var name = this.super(arguments); return name + '::' + 'dog'; } }, Animal ); var dog = new Dog('fluffy', {color: 'white'}); dog.name; // 'fluffy' dog.getSound(); // '...meoww' Dog.getName(); // 'animal::dog'
