ArrayUtils

Package:

com.arthwood.utils

Description

Provides set of methods that operates on any Object instance.
Most of examples show optional use of a method when ArtJs.doInjection() has been performed.

Methods

  • $reject(array:Array, callback:Function):Array

    Works like reject() but modifies array

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.$reject(array, function(i) {return i > 2});
    array // [1, 2]
    // or
    // array.$reject(function(i) {return i > 2});

  • all(array:Array, callback:Function):Boolean

    Returns true if callback for all items returns true

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.all(array, function(i) {return i > 2}); // false
    // or
    // array.all(function(i) {return i > 2});

    ArrayUtils.all(array, function(i) {return i > 0}); // true
    // or
    // array.all(function(i) {return i > 0});

  • any(array:Array, callback:Function):Boolean

    Returns true if callback any passed item returns true

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.any(array, function(i) {return i > 2}); // true
    // or
    // array.any(function(i) {return i > 2});

    ArrayUtils.any(array, function(i) {return i < 0}); // false
    // or
    // array.any(function(i) {return i < 0});

  • arrify(object:Object, idx:Number):Array

    Converts any iterable object into Array starting from idx

    Example:

    function test(value) {
      ArrayUtils.arrify(arguments, 1); // [3, 'hi']
      // or
      arguments.arrify(1); // [3, 'hi'] }

    test(1, 3, 'hi');

  • beforeLast(array:Array):Object

    Returns last but one element of an array

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.beforeLast(array); // 3
    // or
    // array.beforeLast();

  • compact(array:Array):Array

    Returns copy of array without null or undefined items

    Example:

    var array = [1, null, 6, 2, undefined];

    ArrayUtils.compact(array); // [1, 6, 2]
    // or
    // array.compact();

  • detect(array:Array, callback:Function):Object

    Returns first item from array for which callback returns true

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.detect(array, function(i) {return i > 2}); // 3
    // or
    // array.detect(function(i) {return i > 2});

  • each(array:Array, callback:Function):Array

    Iterates over items and passes item to callback

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.each(array, function(value) {alert(value});
    // or
    // array.each(function(value) {alert(value)});

  • eachIndex(array:Array, callback:Function):Array

    Iterates over items and passes index to callback

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.eachIndex(array, function(idx) {alert(idx});
    // or
    // array.eachIndex(function(idx) {alert(idx)});

  • eachPair(array:Array, callback:Function):Array

    Iterates over items and passes index and item to callback

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.eachPair(array, function(idx, value) {alert(idx + value});
    // or
    // array.eachPair(function(idx, value) {alert(idx + value)});

  • isEmpty(array:Array):Boolean

    Returns true if array is empty; false otherwise

    Example:

    var array = [1, null, 6, 2, undefined];

    ArrayUtils.isEmpty(array); // false
    // or
    // array.isEmpty();

    array = [];

    ArrayUtils.isEmpty(array); // true
    // or
    // array.isEmpty();

  • first(array:Array):Object

    Returns first element of an array

    Example:

    var array = [1, 2, 3, 4];

    param ArrayUtils.first(array); // 1
    // or
    // array.first();

  • flatten(array:Array):Array

    Transforms 2-dim array into 1-dim array

    Example:

    var array = [[1, 2], [3, 4]];

    ArrayUtils.flatten(array); // [1, 2, 3, 4]
    // or
    // array.flatten();

  • getItem(array:Array, idx:Number):Object

    Returns element of an array on idx position

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.getItem(array, 2); // 3
    // or
    // array.getItem(2);

  • include(array:Array, item:Object):Boolean

    Returns true if array includes item

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.include(array, 2); // true
    // or
    // array.include(2);

    ArrayUtils.include(array, 7); // false
    // or
    // array.include(7);

  • includeAll(array:Array, subset:Array):Boolean

    Returns true if all items of subset occure in array

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.includeAll(array, [3, 1]); // true
    // or
    // array.includeAll([3, 1]);

    ArrayUtils.includeAll(array, [3, 7]); // false
    // or
    // array.includeAll([3, 7]);

  • indexOf(array:Array, item:Object):Number

    Returns index of first occurence of item in array

    Example:

    var array = [1, 2, 3, 4, 2];

    ArrayUtils.indexOf(array, 2); // 1
    // or
    // array.indexOf(2);

  • inject(array:Array, init:Object, callback:Function):Object

    Iterates over items and passes init, item and index to callback

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.inject(array, function(mem, obj) {return mem + obj}); // 10
    // or
    // array.inject(function(mem, obj) {return mem + obj});

  • insertAt(array:Array, idx:Number, insertion:Array):Array

    Inserts insertion array into array at position specified by idx

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.insertAt(array, 2, ['hello', 'mister']); // [1, 2, 'hello', 'mister', 3, 4]
    // or
    // array.insertAt(2, ['hello', 'mister']);

  • intersection(array:Array):Array

    Expects 2-dim array and returns array of common elements

    Example:

    var array = [[1, 2], [2, 5], [6, 2]];

    ArrayUtils.intersection(array); // [2]
    // or
    // array.intersection();

  • last(array:Array):Object

    Returns last element of an array

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.last(array); // 4
    // or
    // array.last();

  • map(array:Array, callback:Function):Array

    Maps key and value to Array item

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.map(array, function(value, idx) {return value + idx}); // [1, 3, 5, 7]
    // or
    // array.map(function(value, idx) {return value + idx});

  • numerize(array:Array):Array

    Turns all items to numbers.

    Example:

    var array = [1, '6', '2'];

    ArrayUtils.numerize(array); // [1, 6, 2]
    // or
    // array.numerize();

  • partition(array:Array, callback:Function):Point

    Returns point object with x, y containing array of array items for which callback returns true and false respectively

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.partition(array, function(i) {return i > 2}); // ([3, 4], [1, 2])
    // or
    // array.partition(function(i) {return i > 2});

  • print(array:Array):Void

    Prints out the array by using p() for each item.

    Example:

    var array = [1, 6, 2];

    ArrayUtils.print(array);
    // 1
    // 6
    // 2

    // or
    // array.print();

  • reject(array:Array, callback:Function):Array

    Returns copy of array without properties for which callback returns true

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.reject(array, function(i) {return i > 2}); // [1, 2]
    // or
    // array.reject(function(i) {return i > 2});

  • removeAt(array:Array, idx:Number):Array

    Removes item from array at position specified by idx

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.removeAt(array, 2); // [1, 3, 4]
    // or
    // array.removeAt(2);

  • removeItem(array:Array, item:Object, onlyFirst:Boolean):Array

    Removes item from array. If onlyFirst is set to false then all item occurences are removed

    Example:

    var array = [1, 2, 3, 4, 2];

    ArrayUtils.removeItem(array, 2, false); // [1, 3, 4]
    // or
    // array.removeItem(2, false);

    array = [1, 2, 3, 4, 2];

    ArrayUtils.removeItem(array, 2, true); // [1, 3, 4, 2]
    // or
    // array.removeItem(2, true);

  • second(array:Array):Object

    Returns second element of an array

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.second(array); // 2
    // or
    // array.second();

  • select(array:Array, callback:Function):Array

    Returns copy of array containing only items for which callback returns true

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.select(array, function(i) {return i > 2}); // [3, 4]
    // or
    // array.select(function(i) {return i > 2});

  • selectNonEmpty(array:Array):Array

    Expects 2-dim array and returns it's copy without empty arrays

    Example:

    var array = [[1, 2], [], [6, 2]];

    ArrayUtils.selectNonEmpty(array); // [[1, 2], [6, 2]]
    // or
    // array.selectNonEmpty();

  • stringify(array:Array):Array

    Turns all items to strings.

    Example:

    var array = [1, 6, 2];

    ArrayUtils.stringify(array); // ['1', '6', '2']
    // or
    // array.stringify();

  • sum(array:Array):Object

    Calculates the sum of items

    Example:

    var array = [1, 6, 2];

    ArrayUtils.sum(array); // 9
    // or
    // array.sum();

    array = [1, '6', 2];

    ArrayUtils.sum(array); // 162
    // or
    // array.sum();

  • third(array:Array):Object

    Returns third element of an array

    Example:

    var array = [1, 2, 3, 4];

    ArrayUtils.third(array); // 3
    // or
    // array.third();

  • uniq(array:Array, callback:Function):Array

    Removes duplicates from array. You can optionally pass callback as compare function

    Example:

    var array = [1, 2, 3, 1, 4, 3];

    ArrayUtils.uniq(array); // [1, 2, 3, 4]
    // or
    // array.uniq();

    var array = [{id: 3}, {id: 12}, {id: 5}, {id: 3}];

    ArrayUtils.uniq(array, function(i, j) {return i.id == j.id}); // [{id: 3}, {id: 12}, {id: 5}]
    // or
    array.uniq(function(i, j) {return i.id == j.id});
