Fade

Package:

com.arthwood.tween

Description

Implements fade effect

Constructor

  • Fade(element:Element, p1:Number, p2:Number, delta:Number = 0.05, interval:Number = 20)

    element - element on which effect is applied

    p1 - start value

    p2 - end value

    delta - increment

    interval - animation frame interval in miliseconds

Events

  • onFinish(fade:Fade)

    Dispatched after effect finish

Properties

  • element:Element

    The same property as in constructor

  • p1:Number

    The same property as in constructor

  • p2:Number

    The same property as in constructor

  • delta:Number

    The same property as in constructor

  • interval:Number

    The same property as in constructor

Methods

  • start():Void

    Starts the animation

  • stop():Void

    Stops the animation

  • isRunning():Boolean

    Indicates if animation is in progress
