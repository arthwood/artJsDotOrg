View

{{code('View')}} is an abstract class based of {{linkTo('Component', 'component2')}} class that encapsulates data used in order to render the view.

Model

Each {{code('View')}} class instance has its private {{code('_model')}} field which is an instance of {{code('artjs.data.Model')}} class. You can either build your model and assign it to the field, or use public {{code('setModel()')}} method. The latter will cause the view to render.
The main feature of a {{code('View')}} is the fact that whenever you make a change to model, an abstract protected {{code('_render()')}} method is being called. Only specific properties of a model will trigger rendering the view, namely the ones that you register: 

this._model.addProperty('myProp');

One of simple {{code('View')}} implementations is {{linkTo('TemplateView', 'template_view')}}.
