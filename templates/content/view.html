View

View is a Component that couples data in Model with the Template.

Definition

You define View as a class attached to html element: 

<div class="art artjs-View"></div>

View will try to render content of the element (will treat it as a template).
Optionally with data-template="<template_id>" attribute you can provide template id to render as content. Another option is data-replace="true" so the rendered template will replace the element.

In most cases you will create own class that extends View and declare it in HTML:

<div class="art artjs-CustomView" data-template="template_id" ></div>

Model

Each View class instance has its private _model property which is an instance of artjs.data.Model class.
The main feature of a view is the fact that whenever you make a change to model, associated template is automatically re-rendered. Only specific properties of a model will trigger re-rendering the template, namely the ones that you register: 

this._model.addProperty('myProp');

It's also possible to assign another model to view using setModel() method.
