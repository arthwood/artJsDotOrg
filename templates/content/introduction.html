Introduction

{{artjs()}} is a powerful library that allows you to build your web application from bottom to the top. It consists of several features that simplify web development. Those features include:

  • {{linkTo('Utils', 'utils')}} - extend functionality of build in objects
  • {{linkTo('Class', 'class2')}} - class definition with full OOP support
  • {{linkTo('Component', 'component2')}} - encapsulate behaviour in single class
  • {{linkTo('View', 'view')}} - bind model to template
  • {{linkTo('Template', 'template')}} - reuse html with different content
  • {{linkTo('Routing', 'routing')}} - map url to action
  • {{linkTo('Testing', 'testing')}} - includes own testing framework
