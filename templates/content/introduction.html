Introduction

ArtJs is a powerful library that allows you to build your web application from bottom to the top. It consists of several features that simplify web development. Those features include:

  • Utility methods - extend functionality of build in objects
  • Class - class definition with full OOP support
  • Template - reuse html with different content
  • Component - encapsulate behaviour in single class
  • View - bind model to template
  • Test - includes own testing framework
  • Lang - I18n support
