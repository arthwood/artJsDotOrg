TemplateView

{{code('TemplateView')}} is a class based of {{linkTo('View', 'view')}} that responds to render by rendering a template.

Definition

You define {{code('TemplateView')}} as a class attached to html element: 

<div class="art artjs-TemplateView"></div>

{{code('TemplateView')}} will try to render content of the element (will treat it as a template).
Optionally with {{code('data-template="<template_id>"')}} attribute you can provide template id to render as content. Another option is {{code('data-replace="true"')}} so the rendered template will replace the element.

In most cases you create own class that extends {{code('TemplateView')}} and declare it in HTML:

<div class="art artjs-CustomTemplateView" data-template="template_id" ></div>
