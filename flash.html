Flash

Package:

com.arthwood.ui

Description

Displays flash message

Constructor

  • Flash(element:Element, path:String, delay:Number = 6)

    element - element on which will display flash message

    path - path to icons (without trailing slash)

    delay - time before message disappears (in seconds)

    Example:

    var flash = new Flash(element, 'images/flash', 4);

Properties

  • element:Element

    The same property as in constructor

  • path:String

    The same property as in constructor

  • delay:Number

    The same property as in constructor

Methods

  • show(type:String, message:String):Void

    Displays flash message (accepted types: 'info', 'warning', 'error')

    Example:

    var flash = new Flash(element, 'images/flash', 4);

    // ... at some point
    flash.show('info', 'Project created sucessfully!');

  • hide():Void

    Hides flash message

    Example:

    var flash = new Flash(element, 'images/flash', 4);

    // ... at some point
    flash.show('info', 'Project created sucessfully!'); // ... at some point (preferrably when flash is still visible)
    flash.hide();
